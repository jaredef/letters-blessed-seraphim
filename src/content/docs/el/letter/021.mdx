26 Μαρτίου / 8 Απριλίου [1965]

Σύναξη του Αρχαγγέλου Γαβριήλ

Αγαπητέ Γκλέμπ,

Δεν μπορώ να βρω τίποτα για τις διώξεις στην Ποτσάεβ Λάβρα στην Αγγλική Ορθόδοξη Ζωή (υπήρχε κάτι για τους Σιβηριανούς Προτεστάντες αγρότες τον Ιανουάριο του 1963, αλλά αυτό είναι όλο), και δεν έχω κανένα τεύχος στα Ρωσικά πριν το 1963.

Ο Διάβολος έχει αρχίσει το έργο του εναντίον μας. Μια Ρωσίδα ήρθε το πρωί για να μας κοιτάξει και να μας πει για τις φήμες που κυκλοφορούν για εμάς στην ρωσική αποικία: ότι είμαστε κομμουνιστές με ένα κατάστημα γεμάτο σοβιετικά βιβλία; ότι είμαστε σοβιετικοί διπλωμάτες που χρησιμοποιούν το κατάστημα ως κάποιο είδος πρόσοψης; ότι είμαστε Αμερικανοί προσηλυτισμένοι (!); κ.λπ. Μέχρι να ανακαλύψει ότι δεν ήμουν Ρώσος, ήταν τόσο γοητευμένη που δεν την πείραξε πολύ και αγόρασε ακόμη δέκα δολάρια σε αυγά, εικονίδια και κάρτες.

"Όταν έρθεις, πρέπει να σκεφτούμε να παραγγείλουμε μερικά από τα μεγάλα εικονίδια της Τζόρντανβιλ, ακόμα κι αν δεν είναι στην καλύτερη στυλ.

Έβαλα ξανά τη διαφήμιση στην Ρωσική Ζωή για αυτό το Σαββατοκύριακο. Για να διατηρήσουμε τη λίγη φήμη που έχουμε, δεν μπορούμε να διαφημιστούμε σε άλλες εφημερίδες. Επίσης, θα πρέπει να έχουμε τον Σκριπκίν ή κάποιον σαν αυτόν να γράψει ένα άρθρο για εμάς για τη Ρωσική Ζωή.

Προσευχήσου για μένα.

Ο αδελφός σου εν Χριστώ,

Ευγένιος
