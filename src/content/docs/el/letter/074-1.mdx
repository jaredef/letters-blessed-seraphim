Χαιρετισμοί στον Κύριο ημών Ιησού Χριστό. Θα χαρούμε να σας δούμε εσάς και την οικογένειά σας στο ερημητήριο μας οποιαδήποτε στιγμή μπορείτε να έρθετε.

Αποθαρρύνουμε τις επισκέψεις γυναικών αυτή τη στιγμή απλώς και μόνο επειδή δεν είμαστε εξοπλισμένοι γι' αυτές. Αλλά αυτές οι λίγες που καταφέρνουν να ξεπεράσουν τα πάντα και να φτάσουν εδώ είναι ευπρόσδεκτες για μια σύντομη επίσκεψη. Στην πραγματικότητα, μερικές από τις τοπικές κυρίες έχουν ήδη οδηγήσει κατευθείαν στη μέση μας, οπότε δεν υπάρχει λόγος να αρνηθούμε τις Ορθόδοξες γυναίκες την ίδια προνόμιο, τουλάχιστον μια φορά και για λίγο!

Αλεξέι: σας χαιρετούμε στην πρώτη σας γιορτή ονόματος στην Εκκλησία. Είθε ο Θεός και ο προστάτης άγιος σας να σας καθοδηγούν σε όλη σας τη ζωή στον ευθύ και στενό Ορθόδοξο δρόμο!
