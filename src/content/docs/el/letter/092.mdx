16 Αυγούστου/29 Αυγούστου 1971

Η Αγία Πετσέτα

Αγαπητέ Αδελφέ εν Χριστώ, Λαυρέντιε,

Σας ευχαριστώ που περάσατε την εορτή μαζί μας. Οι υπηρεσίες μας για κάποιο λόγο σήμερα κράτησαν 3 ώρες, οπότε πιθανώς ήσασταν στο Γουίλιαμς πριν τελειώσουμε. Με την Ουσπενίγιε περνάμε σε φθινοπωρινό χρόνο, και η ώρα που ξυπνάμε γίνεται προ-χαραυγής, όπως εύστοχα μας υπενθυμίσατε.

Στις τελευταίες ώρες έχουμε σκεφτεί λίγο το πρόβλημά σας και θα προσφέρουμε τους καρπούς του, με τη μορφή μερικών αρχών και συστάσεων, οι οποίες ίσως, όταν συνδυαστούν, θα σας βοηθήσουν στο δρόμο προς μια απόφαση.

[1.] Διατηρήστε το “πνευματικό” και το “κοσμικό” ξεχωριστά: και οι δύο σφαίρες έχουν τη δική τους λογική. Η απόφασή σας να μας ενωθείτε, και αργότερα να παραμείνετε μαζί μας, αν ο Θεός το θελήσει, θα πρέπει να ληφθεί σύμφωνα με την πνευματική λογική· αλλά οι προπαρασκευαστικές ερωτήσεις όπως ταξίδι, εργασία, κ.λπ., υπόκεινται ακόμα σε κοσμική λογική. Είναι επικίνδυνο λάθος να αρχίσετε να εφαρμόζετε πνευματική λογική πολύ νωρίς ή σε λάθος μέρος, διότι αυτό μπορεί να εμποδίσει αργότερα την ασφαλή επιστροφή σας στον κόσμο. Οι περιπλανώμενοι και οι πρώην μοναχοί μας είναι ζωντανά παραδείγματα του τι να μην ακολουθήσετε· αν κάποιος πρόκειται να ζήσει στον κόσμο (και μπορεί να το κάνει καρποφόρα, από χριστιανική άποψη), θα πρέπει να το κάνει με την ανεξαρτησία και τα πλεονεκτήματα οποιουδήποτε άλλου που ζει στον κόσμο, και όχι με το εμπόδιο της πνευματικής αναπηρίας που προέρχεται από το να μην ξέρει πού ανήκει, προσπαθώντας να αναμείξει μοναστική και κοσμική ζωή, έχοντας μετανιώσει για παρελθόντες αποφάσεις, κ.λπ.

Οι μοναστικοί κανόνες δίνουν ακριβώς μια δοκιμαστική περίοδο τουλάχιστον 6 μηνών, κατά τη διάρκεια της οποίας τα κοσμικά δικαιώματα ενός ατόμου, ενώ είναι ανασταλμένα, παραμένουν άθικτα. Επομένως, δεν είναι απλώς κατάλληλο να κάνετε μια οριστική απόφαση τώρα σχετικά με την παραμονή σας μαζί μας. Σας καλωσορίζουμε και ελπίζουμε ότι τελικά θα πάρετε αυτή την απόφαση, καθώς μας φαίνεται ότι θα είστε πιο καρποφόρος εδώ· αλλά αυτή η απόφαση ανήκει στο τέλος της δοκιμαστικής περιόδου, κατά τη διάρκεια της οποίας όλες οι προηγουμένως απροσδιόριστες ερωτήσεις θα πρέπει να επιλυθούν. Επομένως, σίγουρα, αν είναι δυνατόν, θα πρέπει να πάρετε μια άδεια απουσίας, ώστε αν αποφασίσετε να επιστρέψετε στον κόσμο να έχετε τη μέγιστη ελευθερία και ασφάλεια για να φέρετε όποιον καρπό μπορείτε να φέρετε στον κόσμο (το ημερολόγιο είναι ένας καλός πρώτος καρπός) χωρίς να εξαντλήσετε την ψυχική-πνευματική σας ενέργεια σε μάταιες μετανιές και ανασφάλεια στην εργασία.

[2.] Ένα από τα σημαντικά σας χρέη προς τον κόσμο είναι η επίσκεψη στη μητέρα σας, η οποία από ό,τι έχετε πει και τον τρόπο που φαίνεται να αισθάνεστε, θα πρέπει να γίνει αν είναι δυνατόν φέτος.

[3.] Οι πρώτες δύο αρχές φαίνεται να είναι αμοιβαία αποκλειστικές, ίσως θα πρέπει να αλλάξετε κάποιο άλλο στοιχείο στην εξίσωση για να τις κάνετε και τις δύο δυνατές. Τι θα λέγατε για την επίσκεψη στη μητέρα σας, τους συγγενείς και τη Βοστώνη στις διακοπές σας (3 εβδομάδες;) με δανεικά χρήματα που θα επιστρέψετε μετά τα Χριστούγεννα (όταν θα έχει τελειώσει το χρέος σας των $400;), και στη συνέχεια να μας ενωθείτε την άνοιξη αντί για τον Νοέμβριο; Φυσικά, δεν ξέρω ποιες ποσότητες χρημάτων θα εμπλέκονται ή πόσο καιρό θα χρειαστεί να το επιστρέψετε, αλλά το ταξίδι σας θα πρέπει προφανώς να είναι όσο το δυνατόν πιο ταπεινό.

Πέρα από αυτό, δεν ξέρουμε τι άλλο να πούμε ή να προτείνουμε, καθώς δεν βλέπουμε άλλες εναλλακτικές. Προσευχηθείτε γι' αυτό στον Βλαδίκα Ιωάννη.

Ξεχάσαμε να αναφέρουμε: Ο πατήρ Θεοδωρίτος (μοναχός του Άθω, πρώην Ιωάννης Μαυρός, ο οποίος έγραψε άρθρα και μας βοήθησε να ρυθμίσουμε τη γραφή το 1965) επισκέπτεται τώρα τον π. Παντελεήμονα και θα έρθει στο Σαν Φρανσίσκο αυτή την εβδομάδα ή την επόμενη για να περάσει αρκετές εβδομάδες με τον αδελφό του. Θα ήθελε να μας επισκεφθεί, και ο π. Παντελεήμονας του δίνει τον αριθμό τηλεφώνου σας. Παρακαλώ κάντε ό,τι μπορείτε για να τον φέρετε εδώ: δίνοντας οδηγίες (σε περίπτωση που ο αδελφός του μπορεί να τον φέρει), ειδοποιώντας μας πού και πότε να τον παραλάβουμε (θα πηγαίνουμε στο ταχυδρομείο τουλάχιστον κάθε δεύτερη μέρα) σε περίπτωση που έρθει με λεωφορείο ή αεροπλάνο, ή ίσως ακόμη και φέρνοντάς τον εσείς οι ίδιοι (θα πληρώσουμε ευχαρίστως το ταξίδι σας). Έχει γράψει δύο βιβλία στα ελληνικά, για τα οποία ετοιμάζουμε μια μερική ρωσική περίληψη: ένα για τη ζωή του Αγίου Νεκταρίου, ένα άλλο που αποτελείται από διαλόγους κατά του οικουμενισμού. Σε κάθε περίπτωση, θα ήταν καλό να τον γνωρίσετε—ο αδελφός του, Αντώνιος Μαυρός, είναι [στο] τηλέφωνο· καλέστε τον αν δεν σας καλέσει σε μια εβδομάδα ή δύο.

Προσευχηθείτε για εμάς. Και προσέξτε· εκτός από τον ανθρώπινο παράγοντα, ο διάβολος πιθανώς θα προσπαθήσει να σας αναστατώσει ή να σας μπερδέψει τις επόμενες εβδομάδες. Μην εμπιστεύεστε πολύ τα συναισθήματά σας, και κάντε νόημα τουλάχιστον με κοσμική λογική. [Η επιστολή τελειώνει.]
