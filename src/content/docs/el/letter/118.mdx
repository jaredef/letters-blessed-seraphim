1/14 Μαρτίου 1973

Τρίτη ημέρα της Μεγάλης Σαρακοστής

Αγαπητέ Πατέρα Νεκτάριε,

Ευλογείτε! Είμαστε πολύ χαρούμενοι που ακούσαμε την θετική σας απάντηση για την προτεινόμενη περιοδική έκδοση. Είμαστε πολύ ευγνώμονες για την προσφορά σας να την εκτυπώσετε για εμάς. Ωστόσο, η εκτύπωση από μόνη της θα ήταν το λιγότερο από τα έργα μας, καθώς σκοπεύουμε να εκτυπώσουμε 4 σελίδες τη φορά (μικρότερες από το Orth. Word), 16-24 σελίδες, και πιθανώς μόνο 1000 αντίτυπα για αρχή—έτσι η εκτύπωση θα συνεπάγεται μόνο ένα κλάσμα (10% ή κάτι τέτοιο) της εργασίας στην εκτύπωση του OW.

Σε κάθε περίπτωση, πρέπει να μιλήσουμε ξανά για ολόκληρο το έργο με τον Βλάδικα Νεκτάριο, ο οποίος μερικές φορές μπορεί να είναι πολύ διστακτικός για τέτοια πράγματα. Ίσως είναι καλύτερα να μην αρχίσουμε τόσο σύντομα, μόνο να θυμόμαστε ότι, ανεξαρτήτως του τι συμβαίνει, ο Βλάδικας Νεκτάριος μας έδωσε την ευλογία του να αναβιώσουμε την περιοδική έκδοση του Βλάδικα Ιωάννη.

Ωστόσο, θα θέλαμε πολύ να εκτυπώσουμε κάτι για την 7η επέτειο της κοίμησης του Βλάδικα Ιωάννη. Θα θέλαμε επίσης να κάνουμε κάτι για τη μνήμη του Βλάδικα Σάββα. Διαβάζοντας τα έγγραφά του και επίσης τα άρθρα του στην Ορθόδοξη Ρωσία, βλέπουμε ότι είχε ένα σαφές μήνυμα για τον ρωσικό λαό, το οποίο πιθανώς θα ξεχαστεί τώρα αν δεν κάνουμε κάτι γι' αυτό. Τα άρθρα του για τον Βλάδικα Ιωάννη μιλούν πιο δυνατά από οποιονδήποτε από εμάς για το γεγονός ότι είναι ένας άγιος που δεν έχει ακόμη εκτιμηθεί σωστά και που έχει μεγάλη σημασία για τον ορθόδοξο λαό. Η διαθήκη του προς εμάς είναι, προφανώς, να συνεχίσουμε να μιλάμε αυτή την αλήθεια, ακόμη και αν η “οργανωτική” νοοτροπία δεν θέλει να την ακούσει.

Διαβάζοντας τα έγγραφα του Βλάδικα Σάββα, βρίσκουμε να του απευθύνονται οι συνήθεις καταγγελίες κατά του Βλάδικα Ιωάννη (αυτά είναι τα χειρότερα πράγματα που μπορούν να βρουν γι' αυτόν!): ότι είναι ευέξαπτος, αγενής, αδιάφορος προς τους άλλους, (για παράδειγμα επειδή έρχεται αργά στις υπηρεσίες), διαταράσσει τη συνήθη τάξη των πραγμάτων, είναι κακός διαχειριστής, δεν μπορεί να γίνει κατανοητός, μουρμουρίζει και κοιμάται στη μέση των πιο σημαντικών (κοσμικών) συζητήσεων, ότι οι κληρικοί του Καθεδρικού Ναού του Σ.Φ. θεωρούν ότι είναι αργία όταν δεν είναι παρών. Εγώ ο ίδιος είχα την ευκαιρία να είμαι μάρτυρας των περισσότερων από αυτά τα φαινόμενα, και μπορώ να βεβαιώσω, για παράδειγμα, ότι στις πολλές περιπτώσεις που ήταν “αγενής” μαζί μου ήμουν εξαιρετικά ευγνώμον και είδα μόνο πνευματικό όφελος σε αυτό. Σε όλα αυτά νομίζω ότι κρύβεται ένα σημαντικό γεγονός για τον Βλάδικα που δεν έχει αναδειχθεί πολύ: ότι αρνήθηκε να επιτρέψει στην Εκκλησία να γίνει συνήθεια, και με την φαινομενική του αγένεια, προσπάθησε να ξυπνήσει τους ανθρώπους από την πνευματική ρουτίνα στην οποία μπορεί να είναι τόσο εύκολο να πέσουν. Μόλις ο Βλάδικας έφυγε, όλα έγιναν “ομαλά” στον Καθεδρικό Ναό και οι κληρικοί ήταν ικανοποιημένοι—διότι τώρα μπορούσαν και έκαναν την Εκκλησία μια συνήθεια, ανίκανη να εμπνεύσει κανέναν εκτός από το κεφάλαιο του παρελθόντος που δεν έχει ακόμη εξαντληθεί (αλλά θα είναι πριν από πολύ καιρό!).

Αλλά να έρθουμε στο θέμα: θα θέλαμε να εκτυπώσουμε, πριν από τις 2 Ιουλίου, τα άρθρα του Βλάδικα Σάββα για τον Βλάδικα Ιωάννη σε μορφή βιβλίου. Αυτό θα σήμαινε περίπου 100 σελίδες υλικού από την Ορθόδοξη Ρωσία, και λίγο περισσότερο υλικό και μερικές φωτογραφίες που θα μπορούσαμε να εκτυπώσουμε μόνοι μας. Πόσο ρεαλιστική είναι η δυνατότητα να εκτυπώσετε αυτές τις 100 σελίδες (δηλαδή, 50 εκτυπώσεις χαρτιού 8 1/2 x 11, 2000 αντίτυπα); Με ποιο ρυθμό μπορεί να γίνει η εργασία και πόσο, για παράδειγμα, θα μπορούσε να γίνει αν κάποιος από εμάς (ή ίσως και οι δύο) ερχόταν και βοηθούσε για το μεγαλύτερο μέρος μιας εβδομάδας; Αν αυτή η πρόταση δεν σας κατακλύσει εντελώς, παρακαλώ ενημερώστε μας για κάποιους από τους τεχνικούς και χρονικούς παράγοντες που εμπλέκονται. Ποιο είναι το μεγαλύτερο φύλλο που μπορεί να πάρει η μηχανή σας ή να φτιάξει ο κατασκευαστής πλάκας σας; (Αν ήταν 11×17 θα επιτάχυνε τα πράγματα, αλλά αμφιβάλλουμε γι' αυτό.)

Όπως βλέπετε, θα θέλαμε πολύ να βγούμε με κάτι τώρα που θα είναι και για τη δόξα του Βλάδικα Ιωάννη και τη μνήμη του Βλάδικα Σάββα, και να το κάνουμε ως κοινό φόρο τιμής από εσάς και εμάς θα μας ευχαριστούσε ακόμη περισσότερο.

Όλα αυτά, φυσικά, είναι “μυστικά,” και δεν το έχουμε αναφέρει καν στον Βλάδικα Νεκτάριο. Είναι πιθανώς καλύτερα για αυτόν αν κρατηθεί “απομακρυσμένος” ούτως ή άλλως! Αν και ο κυβερνών επίσκοπός μας πιθανώς δεν θα είναι ευχαριστημένος με αυτό, δεν νομίζουμε ότι θα πει τίποτα, αφού τελικά είναι μόνο μια “επανέκδοση.” (Αν ενημερωνόταν εκ των προτέρων, φυσικά, θα επέμενε να συμβουλεύει, να αποθαρρύνει, κ.λπ., και στο τέλος να καταστρέψει όλη την ιδέα.)

Ανυπομονούμε, ξανά, για την απάντησή σας σε όλα αυτά. Παρακαλώ προσευχηθείτε για εμάς.

Με αγάπη εν Χριστώ τον Σωτήρα μας,

Σεραφείμ, μοναχός
