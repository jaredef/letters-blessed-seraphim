29 Ιουνίου/12 Ιουλίου, 1973

Άγιοι Απόστολοι Πέτρος και Παύλος

Αγαπητέ Αδελφέ εν Χριστώ, Αλέξιε,

Χαιρετισμούς στον Κύριο ημών Ιησού Χριστό.

Λάβαμε σήμερα την "Ανοιχτή Επιστολή" του π. Εφραίμ, μαζί με το σημείωμά σας. Ναι, βρήκαμε κι εμείς ότι έχασε εντελώς το νόημα, και όλα τα πολύ ωραία αποσπάσματα από τους Πατέρες για τους διαφορετικούς βαθμούς γνώσης δεν σημαίνουν τίποτα όταν κάποιος σταματά να συνειδητοποιήσει ότι δεν επιτίθεστε καθόλου στην επιστημονική γνώση, αλλά μόνο στην ψευδο-επιστημονική ψευδο-θρησκευτική φιλοσοφία που μεταμφιέζεται ως επιστήμη, και χρησιμοποιείτε την επιστημονική γνώση όχι για να υπερασπιστείτε τη θεολογία αλλά μόνο για να καταστρέψετε τις αυτοαναιρούμενες θεωρίες των ψευδο-επιστημόνων. Είναι αυτά τα σημεία πραγματικά τόσο δύσκολο να κατανοηθούν ή αδύνατο να υπερασπιστούν;

Εκτός αν είμαστε τρελοί, ο π. Εφραίμ έχει χαθεί κάπου στα σύννεφα και δεν είναι καθόλου σε επαφή με το τι συμβαίνει στον κόσμο σήμερα, διανοητικά—κάτι που είναι πολύ σημαντικό για τους Ορθόδοξους Χριστιανούς που ζουν σε αυτόν τον κόσμο. Ο π. Εφραίμ, προσπαθώντας να σταθεί τόσο "πάνω" από όλο το ζήτημα, δεν δίνει την εντύπωση ότι μιλάει από τα ύψη του τρίτου βαθμού γνώσης (που φαίνεται να είναι αυτό που προσπαθεί να υπονοήσει;), αλλά μάλλον χρησιμοποιεί αυτή την υψηλή γνώση για μάλλον ορθολογιστικούς σκοπούς. Είμαστε πολύ απογοητευμένοι που βλέπουμε τέτοια στενότητα. Και η απάντηση της μητέρας στον Ορθόδοξο "μαθητή λυκείου" (σελ. 17—αυτό προφανώς συνοψίζει την "απάντησή" του στο όλο πρόβλημα της εξέλιξης!)—πόσο αφελής και άτολμη! Μπορεί ο π. Ε. να είναι πραγματικά τόσο ανίδεος για τους αντι-Χριστιανικούς σκοπούς μιας τέτοιας "επιστημονικής" εκπαίδευσης; Η απάντησή του είναι μια ανοιχτή πρόσκληση στον νέο να δεχτεί ό,τι του διδάσκει το σχολείο—επειδή εμείς οι φτωχοί Ορθόδοξοι Χριστιανοί, αλίμονο, έχοντας τέτοια υψηλή γνώση, δεν μπορούμε να "ξέρουμε καλύτερα." Θα σας πω ειλικρινά (αλλά μην με αναφέρετε!)—αυτό δεν είναι θεολογία, είναι ανοησίες. Υπάρχει ένα πραγματικό και επείγον πρόβλημα εδώ, και το έχετε αντιμετωπίσει τίμια, φρέσκα και καλά (λαμβάνοντας υπόψη τον περιορισμένο χώρο που είχατε); αν υπάρχουν ελλείψεις ή λάθη σε αυτά που είπατε, μπορούν να συζητηθούν με φιλικό τρόπο. Αλλά αλίμονο, ο μόνος του στόχος είναι να σας δυσφημίσει και να σας βάλει στη θέση σας. Αυτό είναι λάθος και άρρωστο. Συγχωρέστε τη δυνατή μου γλώσσα.

Η πρώτη μας σκέψη ήταν—μια απάντηση είναι άσκοπη. Ωστόσο, αν γράφετε μία, θα χαρούμε να τη διαβάσουμε για μια "αντικειμενική" (ή τουλάχιστον τρίτου προσώπου) αξιολόγηση. Ωστόσο, μια απάντηση σημείο προς σημείο είναι πραγματικά άσκοπη. Παρά τις διαβεβαιώσεις του π. Ε., προφανώς δεν έχει καμία συμπάθεια για εσάς και δεν έχει καμία ιδιαίτερη επιθυμία να μάθει τι προσπαθείτε να πείτε—ήδη "ξέρει καλύτερα," και όλα όσα λέτε θα ερμηνευτούν υπό το φως αυτής της "γνώσης." Κάθε καλό σημείο που μπορείτε να κάνετε στην απάντηση θα "επιλεγεί" και θα κομματιαστεί για να αποδειχθεί ότι είστε ένας αναξιόπιστος προσήλυτος και θεολογικά αβάσιμος. Δεν νομίζω ότι αυτό είναι σκόπιμο από μέρους του—απλώς έχει μια προκατειλημμένη ιδέα και δεν πρόκειται να την αλλάξει.

Δυστυχώς, οι εμφανείς Καθολικές αναφορές στο "άρθρο του Σάβανου" δίνουν στον π. Ε. όλα τα πυρομαχικά που χρειάζεται για να αποδείξει ότι είστε ακόμα "Καθολικός," και θα ερμηνεύσει όλα όσα λέτε υπό αυτό το φως. Το κύριο άλλο αδύναμο σημείο σας στο οποίο επικεντρώνεται είναι η υπερέμφαση (έτσι μας φαίνεται) στην επιστήμη στο άρθρο του "Σάβανου". Εκτός από αυτά τα σημεία, δεν βλέπουμε ότι έχει οποιαδήποτε νόμιμη αιτία για μια τέτοια εχθρική και δημόσια επιστολή—στην πραγματικότητα, σε κάθε περίπτωση δεν υπάρχει αιτία για τίποτα περισσότερο από ένα φιλικό και ανήσυχο σημείωμα.

Είναι μάλλον απογοητευτικό να μας πετάνε τον Καλομοίρο ξανά,—ακόμα χωρίς να ξέρουμε τι λέει για την εξέλιξη! Υποψιάζομαι έντονα ότι στην πραγματικότητα δεν λέει τίποτα στο οποίο κανείς από εμάς θα είχε ιδιαίτερη αντίρρηση και ότι δεν πιστεύει καθόλου στην "εξέλιξη" που επιτίθεστε. Παίρνουμε την εντύπωση ότι ο π. Ε. δεν είναι αρκετά ενήμερος για οποιοδήποτε είδος εξέλιξης για να διακρίνει τη διαφορά, αλλά χρησιμοποιεί μόνο τον Καλομοίρο ως ένα βολικό όπλο εναντίον σας.

Μετά την ανάγνωση της επιστολής του π. Ε., αρχίζουμε να απελπιζόμαστε για την "Ελληνική σοφία" της εποχής μας, που φαίνεται να έχει πάρα πολλά κοινά με την αρχαία ποικιλία! Τι θα συμβεί όταν αρχίσουν πραγματικά να ανακαλύπτουν πόσο απλοί και αδιάφοροι για όλα αυτά τα πράγματα που τους ενθουσιάζουν είμαστε εμείς οι "Ρώσοι"; Ένα βασικό στοιχείο φαίνεται να λείπει από όλη τους τη "σοφία," ένα που οι Άγιοι Πατέρες τονίζουν ότι είναι απαραίτητο για την γνήσια Ορθόδοξη ζωή: η δοκιμασία. Η "σοφία" που γεννιέται από την αργία και τις άσκοπες διαμάχες δεν αξίζει να την έχει κανείς; αλλά η σοφία που γεννιέται από τη βαθιά δοκιμασία (όπως ο Θεός έχει δώσει πάνω απ' όλα στους Ρώσους στις μέρες μας) είναι μόνη πραγματικά ισορροπημένη και υγιής, ακόμα κι αν δεν μπορεί να δώσει μια εύκολη απάντηση σε κάθε χλευαστική ερώτηση. Ας προσπαθήσουμε να εισέλθουμε πιο βαθιά σε αυτή τη δοκιμασία, με τη χάρη του Θεού να μας το επιτρέψει!

Ένα αποκαλυπτικό σημείο: η χρήση του "βασιλικού πολτού" σε ένα υποτίθεται σοβαρό θεολογικό άρθρο είναι ασυγχώρητη ελαφρότητα. Ένας Ρώσος θεολόγος μπορεί περιστασιακά να φέρει στο σπίτι κάποιο πολεμικό σημείο με χιούμορ—αλλά με ένα νηφάλιο χιούμορ, όχι απλή ελαφρότητα.

Ήδη εισέρχεστε κάπως σε αυτή τη δοκιμασία μέσω του άδικου χλευασμού. Είθε ο Θεός να σας δώσει τη δύναμη να αντέξετε και να μην αντεπιτεθείτε με τα ίδια όπλα, αλλά μάλλον να συνεχίσετε να κοιτάτε φρέσκα (και όχι αφηρημένα) τα πραγματικά προβλήματα που αντιμετωπίζουν σήμερα όσοι επιθυμούν να είναι Ορθόδοξοι σε έναν κόσμο που έχει τρελαθεί.

Οποιαδήποτε απάντηση κάνετε στον π. Ε. θα πρέπει να είναι σύντομη και στο θέμα. Προφανώς έχει εκμεταλλευτεί άδικα την κατάστασή σας για να σας δυσφημίσει πλήρως, βασιζόμενος στη φήμη του μοναστηριού εναντίον εσάς, ενός "κανενός." Ιππεύει σε ένα ρεύμα διανοητικής μόδας, και αυτό θα περάσει, και δεν θα είναι για το καλό του μοναστηριού που επέτρεψε στον εαυτό του να το κάνει αυτό και να μην αντιμετωπίσει τα πραγματικά διανοητικά προβλήματα της εποχής. Προσευχηθείτε στον Βλαδίκα Ιωάννη για καθοδήγηση. Να ξέρετε ότι δεν εξαρτώνται όλα από το τι "σκέφτονται" κάποιοι για εσάς, και επίσης ότι στον κατάλληλο χρόνο άλλοι θα μιλήσουν για εσάς.

Είστε ευπρόσδεκτος να μας επισκεφθείτε το επόμενο Σαββατοκύριακο, ή όποτε μπορείτε να έρθετε.

Με αγάπη στον Χριστό τον Σωτήρα μας,

Υ.Γ. Αναμφίβολα θα ήταν καλύτερο να μην πείτε στον π. Εφραίμ ότι εργάζεστε σε μια μεγαλύτερη μελέτη για την εξέλιξη—αφήστε το να έρθει ως πλήρης έκπληξη, αλλιώς θα προσπαθήσει με κάθε τρόπο να το υπονομεύσει πριν βγει.

Δεν θα είναι εύκολο να περάσετε το νόημα της μεγαλύτερης μελέτης σας για την εξέλιξη σε ανθρώπους που σκέφτονται όπως ο π. Εφραίμ, αλλά με τη βοήθεια του Θεού μπορεί να γίνει. Υπάρχει κάτι βαθύ και σημαντικό εδώ—μια μάλλον "ακαδημαϊκή" προσέγγιση στη θεολογία που δεν έρχεται σε επαφή με την αντι-θεολογία της εποχής μας.
