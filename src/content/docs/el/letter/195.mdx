22 Οκτωβρίου/4 Νοεμβρίου 1975

Μητέρα του Θεού Καζάν

Ισαπόστολος Αβέρκιος

Αγαπητέ Αλεξέι,

Χαιρετισμούς στον Κύριο ημών Ιησού Χριστό, και συγχαρητήρια για την ονομαστική εορτή του Βλαδίκα Αβέρκιου! Ρωτήσαμε μόλις τον π. Μιχαήλ Πομαζάνσκι: ποιος είναι ο αυθεντικός Ορθόδοξος θεολογικός οδηγός για σήμερα, και η απάντησή του: Αρχιεπίσκοπος Αβέρκιος. Ο π. Μιχαήλ μας είπε μερικά άλλα σημαντικά πράγματα που θα μοιραστούμε μαζί σας όταν σας δούμε. (Πότε;)

Σχετικά με τους Παλαιοημερολογίτες: οι παρατηρήσεις σας (και της Νίνας) φαίνονται νηφάλιες και δίνουν την εικόνα τους που περιμέναμε. Σίγουρα δεν υπάρχει καμία βλάβη στη συνεχιζόμενη προσωπική επαφή μαζί τους—αλλά όπως σωστά βλέπετε, μην σκεφτείτε καν να είστε κάποιος είδος μεσολαβητή μεταξύ τους και της Συνόδου. Αν θέλουν να γίνουν Ορθόδοξοι, θα προκύψει στην ώρα τους· αν θέλουν να σας υιοθετήσουν τις παλαιοημερολογίτικες συνήθειες, θα πρέπει να ξέρουν ότι είναι χάσιμο χρόνου. Ο Τζον Χουδάνις φαίνεται να είναι ο τύπος του ενθουσιώδους μεταστραφέντα—ο οποίος θα πέσει εύκολα σε απογοήτευση όταν τα όνειρά του διαψευστούν! Με αυτόν είναι καλύτερο να είστε όσο το δυνατόν πιο ανοιχτοί και να μην τον αφήσετε να κρατάει ψευδαισθήσεις που θα εκραγούν από την καθημερινή πραγματικότητα.

Ο αδελφός Λόρενς επέστρεψε ασφαλής και, αφού το σκέφτηκε, θέλει να “δοκιμάσει ξανά” να μείνει. Τι να πούμε; Μέχρι τώρα είναι προφανές ότι απλώς δεν επιδιώκει αυτό που επιδιώκουμε, αλλά απλώς ακολουθεί, και είναι συναισθηματικά συνδεδεμένος μαζί μας· μισούμε απλώς να του πούμε να βρει κάπου αλλού όπου μπορεί να είναι σε ένα μυαλό και ψυχή (γνωρίζοντας ότι ποτέ δεν θα βρει τέτοιο μέρος), και μισούμε επίσης να τον “σέρνουμε” μαζί μας, απαιτώντας να ενεργεί με τρόπο που πρέπει να είναι εθελοντικός για να λειτουργήσει. Οι “παροξυσμοί” του μερικές φορές (όπως για παράδειγμα αυτή τη φορά) προκαλούν μια τεράστια εξάντληση σε μένα, και στον π. Ερμάν επίσης, γιατί στην πραγματικότητα αυτό που κάνει είναι να αμφισβητεί ολόκληρο τον τρόπο ζωής μας, χωρίς να τον βλέπει ή να τον εκτιμά· και αυτός ο τρόπος ζωής είναι ήδη τόσο δύσκολος, απαιτώντας μεγάλη αγώνα και ευαισθησία, που μπορεί να καταστραφεί. Θέλει έναν “ιδιορυθμικό” τρόπο—δηλαδή, παλαιοημερολογισμό—και απλώς δεν μπορεί να δει ότι αυτό είναι γελοίο και τραγικό όταν έχουμε κάποιον όπως τον π. Ερμάν που μπορεί να μας καθοδηγήσει σε ενότητα ψυχής και νου (κάτι που εμείς οι φτωχοί Αμερικανοί δεν μπορούμε να κάνουμε, όσο η Ορθοδοξία είναι τόσο αδύναμη σε εμάς). Όλοι οι άλλοι αδελφοί που έχουν μείνει εδώ για οποιαδήποτε διάρκεια έχουν δει αυτό, τουλάχιστον σε κάποιο βαθμό· αλλά ο αδελφός Λ. φαίνεται να παραμένει κλειστός σε αυτό. Πιθανώς θα μείνει για λίγο ακόμα σε “δοκιμασία,” για να δει αν μπορεί να κατανοήσει τουλάχιστον λίγο από αυτό που επιδιώκουμε. Προσεύχομαι μόνο να μην φέρει η “μαλακότητά” μας κακό σε κανέναν. Ιδιαίτερα λυπούμαστε που πρέπει να εμπλακείτε σε αυτό, με όλα τα προβλήματά σας ήδη. Ο αδελφός Λ. αισθάνεται τη ζεστασιά της κοινότητάς σας, αλλά και πάλι, δεν θα μπορούσε ποτέ να θυσιαστεί αρκετά για να είναι πραγματικό μέρος μιας τέτοιας κοινότητας. Αν ποτέ θελήσει να έρθει και να ζήσει μαζί σας—προσέξτε, και μην του κάνετε “ειδικές προϋποθέσεις.” Γενικά, θα έχετε αρκετά προβλήματα με ανθρώπους που “νομίζουν ότι μπορούν να ενταχθούν” αλλά δεν είναι πρόθυμοι να κάνουν τη σκληρή δουλειά που απαιτείται γι' αυτό.

Έχουμε την εντύπωση ότι η Νίνα είναι αναστατωμένη μαζί μας για τα “σκληρά λόγια” μας σχετικά με τον π. Π. Πραγματικά νομίζει ότι αυτό είναι κάποια μορφή “προσωπικής αντιπαράθεσης”; Φαίνεται ότι δεν έχει νόημα να προσπαθήσουμε να την “πειστούμε” ότι ο π. Π. είναι “λάθος” σχετικά με οτιδήποτε. Η καλύτερη ασφάλειά σας ενάντια σε οποιαδήποτε ανησυχία μπορεί να σας προκαλέσει είναι η ήρεμη συνείδησή σας ότι ο π. Π. & Co. δεν είναι καμία αρχή για εσάς· έχετε τη δική σας εργασία να κάνετε και αυτός δεν έχει καμία σχέση με αυτό. Αν προσπαθήσει να σας εμπλέξει στο να σκέφτεστε για τον π. Π. καθόλου—απλώς μεταφέρετε αυτή την ιδέα, ήρεμα και σταθερά. Δεν νομίζω ότι ξέρει τι εννοώ όταν λέω ότι θα ήταν καλύτερα να “σπάσει” μαζί του—δεν εννοώ “προσωπικούς δεσμούς,” φυσικά· εννοώ συναισθηματική εξάρτηση, τον “φόβο του μεταστραφέντα” ότι αν δεν είστε μαζί του μπορεί να είστε “λάθος.”

Ο Βλαδίκα Αβέρκιος, παρεμπιπτόντως, μας απάντησε, ευχαριστώντας μας για την “ηθική υποστήριξη” μας σε μια πολύ δύσκολη περίοδο για αυτόν. [η επιστολή τελειώνει]
