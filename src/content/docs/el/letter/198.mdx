25 Οκτωβρίου/7 Νοεμβρίου 1975

Μάρτυρες Μαρκελιανός και Μαρτυρίος

Αγαπητέ Αδελφέ εν Χριστώ, Θωμά,

Χαιρετισμούς στον Κύριο ημών Ιησού Χριστό. Σας ευχαριστώ για την επιστολή σας, στην οποία βλέπουμε την ειλικρινή σας επιθυμία να υπηρετήσετε τον Χριστό, αν ο Θεός το θελήσει, στη μοναστική ζωή.

Είστε ευπρόσδεκτος να μας επισκεφθείτε για κάποιο διάστημα και να δείτε τη ζωή μας από κοντά. Μετά από αυτό θα μπορούσαμε να μιλήσουμε για την παραμονή σας για να γίνετε novice. Δεν ξέρω τι είδους παραμύθια έχετε ακούσει για εμάς, αλλά στην πραγματικότητα η ζωή μας δεν είναι πολύ διαφορετική από αυτή των άλλων μοναστηριών στην Ρωσική Εκκλησία του Εξωτερικού, οι κύριες διαφορές ίσως να είναι στην απομόνωσή μας από τον κόσμο, ειδικά το χειμώνα, και στις κάπως πρωτόγονες συνθήκες διαβίωσης, οι οποίες μπορεί να είναι λίγο δύσκολες το χειμώνα. Ο χειμώνας μας συνήθως δεν είναι πολύ κρύος, αλλά συνήθως έχει αρκετό χιόνι. Επίσης, θα πρέπει να σας πω ότι δεν έχουμε τη Θεία Λειτουργία συχνά, καθώς δεν έχουμε ιερέα που να διαμένει μαζί μας. Ο εξομολόγος μας μας επισκέπτεται αρκετές φορές το χρόνο, και περιστασιακά κάποιος άλλος ιερέας ή ένας από τους δύο επισκόπους μας στην Δυτική Ακτή θα μας επισκεφθεί και θα λειτουργήσει· αλλά μερικές φορές πρέπει να βγούμε έξω για να λάβουμε την Θεία Κοινωνία στην πλησιέστερη εκκλησία (200 μίλια μακριά).

Το πρώτο και πιο σημαντικό πράγμα στη μοναστική ζωή είναι η εμπιστοσύνη, η οποία αποκτάται μέσω της εμπειρίας. Από την επίσκεψή σας (αν ακόμα θέλετε να έρθετε) θα μπορείτε να δείτε αν μπορείτε να τοποθετήσετε την εμπιστοσύνη σας σε εμάς για να σας καθοδηγήσουμε στη μοναχισμό, σύμφωνα με τις διδασκαλίες των Αγίων Πατέρων.

Είστε ευπρόσδεκτος να φέρετε τις εικόνες και τα βιβλία σας, αλλά δεν ξέρω αν θα υπάρχει χώρος για μια ραφιέρα εικόνων ή όχι, καθώς μπορεί να μην έχετε κανονικό κελί στην αρχή.

Για να έρθετε σε εμάς θα πρέπει να πάτε με Greyhound στο Ρέντινγκ, 230 μίλια βόρεια του Σαν Φρανσίσκο και 40 μίλια από εμάς. Θα πρέπει να μας γράψετε (ή να στείλετε ένα mailgram) αρκετές ημέρες νωρίτερα για να βεβαιωθείτε ότι θα ξέρουμε πότε έρχεστε (δεν έχουμε τηλέφωνο), και θα έρθουμε να σας παραλάβουμε.

Αν μετά από όλα αυτά ακόμα επιθυμείτε να μας επισκεφθείτε, θα ήταν καλό να σταματήσετε στον δρόμο σας προς εμάς στο Σεπούλκο του Αρχιεπισκόπου Ιωάννη Μαξιμόβιτς, ένα αληθινό ιερό μέρος (στο υπόγειο του Καθεδρικού Ναού του Σαν Φρανσίσκο στην λεωφόρο Γκεάρι) και να προσευχηθείτε σε αυτόν να σας καθοδηγήσει σωστά στην οδό της σωτηρίας. Είναι ο ισχυρός μεσίτης μας ενώπιον του Θεού.

Παρακαλώ ενημερώστε μας για τα σχέδιά σας και γράψτε μας για οποιαδήποτε ερώτηση μπορεί να έχετε.

Με αγάπη στον Χριστό,

Σεραφείμ, μοναχός
