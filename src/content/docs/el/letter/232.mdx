27 Ιουλίου/9 Αυγούστου 1976[^1]

Αγαπητέ Πατέρα Ιωάννη,

Ευλογείτε!

Σας ευχαριστώ για την “άμεση επιστολή.” Για όλα όσα είπαμε και κάναμε για να προκαλέσουμε αυτή την αντίδραση σε εσάς, ζητούμε ειλικρινά τη συγχώρεσή σας. Ωστόσο, πρέπει επίσης να πω ότι δεν είναι όλα όσα γράψατε εντελώς δίκαια. Αλλά ας υπάρχει ειρήνη μεταξύ μας!

Παρακαλώ μην κάνετε καμία ειδική προσπάθεια να μας στείλετε κάποιον. Ζητήσαμε να “δανειστούμε” τον Μακάριο, όχι πρώτα απ' όλα επειδή είμαστε λίγοι, αλλά ως ειδική περίπτωση, σε απάντηση στην προσφορά που μας έκανε. Εφόσον νομίζετε ότι αυτό είναι ακατάλληλο, η υπόθεση είναι κλειστή για εμάς, και δεν αναζητούμε “επιπλέον χέρια” οπουδήποτε. Όπως είναι, έχετε δύο από τους πρώην “εργάτες” μας στον π. Λαυρέντιο, ο οποίος εμπιστεύομαι ότι δείχνει τον εαυτό του ως σκληρό εργάτη όπως ήταν για εμάς. Αν κάποιος εκφράσει ενδιαφέρον για εμάς, αρκεί να τους δώσετε μια λογικά αντικειμενική εικόνα, χωρίς να τονίσετε ότι χρειαζόμαστε χέρια. Είμαστε ικανοποιημένοι με ό,τι μας στέλνει ο Θεός.

Σχετικά με τον Μητροπολίτη Φιλάρετο, έχουμε λάβει μια μέτρια απάντηση, περισσότερο υπέρ παρά κατά. Η “κατά” απάντηση αποκαλύπτει μια εκπληκτική έλλειψη πληροφόρησης σχετικά με το τι σκέφτονται οι επίσκοποί μας, που ίσως από μόνη της δικαιολογεί την αναφορά του θέματος. Φυσικά, έχετε δίκιο ότι απαιτείται διακριτικότητα σε αυτόν τον τομέα.

Συνημμένη είναι μια επιστολή που είναι αυτοεξηγητική. Έχουμε απαντήσει συνοπτικά σε αυτόν για τον εαυτό μας.

Ζητούμε τις προσευχές σας.

Με αγάπη στον Χριστό,

Σεραφείμ, μοναχός
