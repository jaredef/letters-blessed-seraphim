9 Φεβρουαρίου/22 Φεβρουαρίου 1979

Αγαπητέ π. Γκούντριτζ,[^1]

Σας ευχαριστώ για την επιστολή σας, εκφράζοντας την αγωνία σας για την παρούσα κατάσταση της Αγγλικανικής Εκκλησίας. Ο π. Ερμάν με καθοδήγησε να σας απαντήσω.

Υπάρχει πραγματικά μόνο μία απάντηση στην κατάσταση σας: να ενωθείτε με την Ορθόδοξη Εκκλησία. Αυτό, φυσικά, δεν θα λύσει όλα τα προβλήματά σας, αλλά τουλάχιστον θα σας τοποθετήσει σε μια θέση όπου μπορείτε να τα αντιμετωπίσετε με τα πόδια σας σε στέρεο έδαφος.

Η Ορθόδοξη Εκκλησία μας επίσης, δυστυχώς, υποφέρει από την αναταραχή των καιρών, και τώρα οι περισσότερες Ορθόδοξες δικαιοδοσίες εκφράζουν την προθυμία τους να ακολουθήσουν το “πνεύμα των καιρών.” (αν και, σε σύγκριση με τις Δυτικές ομολογίες, είναι ακόμα αρκετά “πίσω,” δόξα τω Θεώ). Η Ρωσική Εκκλησία εκτός Ρωσίας, ωστόσο, έχει σαφώς πάρει θέση κατά των συμβιβασμών με τους καιρούς, για τον λόγο αυτό θεωρείται μάλλον “πίσω” από τις άλλες Ορθόδοξες δικαιοδοσίες. Θα ήταν καλό να έρθετε σε επαφή με έναν από τους ιερείς μας στο Λονδίνο (έναν μεταστραμμένο στην Ορθοδοξία), ο οποίος θα μπορούσε να συζητήσει μαζί σας την όλη κατάσταση της Ορθόδοξης Εκκλησίας:

π. Ιβς Ντυμπουά

25 Οδός Έμιλι

Λονδίνο W12 9TF

Όσο είστε στην Αγγλικανική Εκκλησία, δεν μπορούμε να κάνουμε πολλά για να σας καθοδηγήσουμε. Θα σας συμβούλευα μόνο να προσεύχεστε θερμά στον Θεό να σας δείξει τον τρόπο να είστε καρποφόροι στην Αληθινή Του Εκκλησία. Οι προσευχές ενός πρόσφατου αγίου της Εκκλησίας μας, του Αρχιεπισκόπου Ιωάννη Μαξίμοβιτς, είναι επίσης πολύ χρήσιμες. Σας στέλνω μία από τις πρόσφατες δημοσιεύσεις μας γι' αυτόν. Μη διστάσετε να γράψετε οποιαδήποτε ερώτηση.
