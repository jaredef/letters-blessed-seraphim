1/14 Μαΐου 1982

Προφήτης Ιερεμίας

Αγαπητέ Αδελφέ εν Χριστώ, Βλαντίμιρ,

ΑΛΗΘΩΣ ΧΡΙΣΤΟΣ ΑΝΕΣΤΗ!

Είθε η ευλογία του Κυρίου να είναι μαζί σας!

Σε απάντηση της επιστολής σας, θα πρέπει να πω πρώτα απ' όλα ότι πιθανόν έχετε μια λανθασμένη άποψη σχετικά με τη φύση και την έκταση των θεολογικών μας μαθημάτων. Δεν έχουμε τίποτα που να μοιάζει με “σεμινάριο,” και τα λίγα μαθήματα που έχουμε προήλθαν αποκλειστικά σε απάντηση της τοπικής ανάγκης να επεκταθεί η θεολογική συνείδηση των αδελφών στο μοναστήρι μας. (Έχουμε καλοκαιρινά μαθήματα σε συνεργασία με την προσκύνηση του π. Ερμάνου, αλλά αυτό διαρκεί μόνο μια εβδομάδα.)

Επομένως, αν ψάχνετε για ένα κανονικό σεμινάριο, με επίσημα αναγνωρισμένα μαθήματα και πτυχίο BTh στο τέλος του, θα ήταν καλύτερα να απευθυνθείτε στο Σεμινάριο της Αγίας Τριάδας στο Τζόρντβιλ. Είναι αλήθεια ότι η κατεύθυνση εκεί είναι πολύ ρωσική, και τα ανώτερα μαθήματα είναι ακόμα κυρίως στη ρωσική γλώσσα, αλλά τα πρώτα δύο χρόνια είναι τώρα κυρίως στα αγγλικά και με κάποια προσπάθεια στη μελέτη της ρωσικής θα μπορούσατε να ολοκληρώσετε ολόκληρο το μάθημα.

Έχοντας πει αυτό, θα προσθέσω ότι έχουμε είχε κάποιους μη μοναχούς αδελφούς να συμμετάσχουν στα μαθήματά μας, και είναι πιθανό να θελήσετε να συμμετάσχετε σε αυτούς. Ωστόσο, η προσέγγισή μας σε αυτά τα μαθήματα είναι λιγότερο προσανατολισμένη στη formal education και περισσότερο στη διαμόρφωση μιας Ορθόδοξης νοοτροπίας—αποβάλλοντας κάποιες από τις λανθασμένες στάσεις που αποκτώνται από την ατμόσφαιρα της σύγχρονης κοινωνίας μας, αναπτύσσοντας μια υγιή στάση προς τον πολιτισμό, την τέχνη, τη λογοτεχνία, τη μουσική, κ.λπ., που θα επιτρέψει σε κάποιον να προσεγγίσει τη θεολογία όχι σε ένα κενό, όπως συμβαίνει τόσο συχνά. Οι μη μοναχοί αδελφοί μας συμμετέχουν πλήρως στην καθημερινή ζωή του μοναστηριού και υπόκεινται σε αυστηρή πειθαρχία (χωρίς “Σαββατοκύριακα ελεύθερα για να πάνε στην πόλη”). Τα μαθήματά μας για το επόμενο φθινόπωρο δεν είναι ακόμη καθορισμένα και θα εξαρτηθούν από το ποιοι θα είναι εδώ για να τα παρακολουθήσουν.

Αν ενδιαφέρεστε για κάτι τέτοιο, μπορεί να μπορέσουμε να σας βοηθήσουμε, αλλά ο μόνος τρόπος για να το διαπιστώσετε σίγουρα είναι να έρθετε και να μας επισκεφθείτε για λίγο, όταν θα γίνει προφανές αν μπορείτε να ενταχθείτε στη ζωή μας και να ωφεληθείτε από αυτήν. Είστε πολύ ευπρόσδεκτος να μας επισκεφθείτε αυτό το καλοκαίρι. (Πρέπει να προσθέσω ότι οι συνθήκες διαβίωσής μας είναι λίγο πρωτόγονες, με λίγα σε σύγχρονες ανέσεις όπως υδραυλικά, ηλεκτρικό ρεύμα, κ.λπ.)

Μη διστάσετε να γράψετε οτιδήποτε άλλο σχετικά με τις ελπίδες, τα σχέδιά σας, κ.λπ.

Με αγάπη εν Χριστώ,

Ανάξιος Ιερομόναχος Σεραφείμ
