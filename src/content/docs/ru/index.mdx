## Настоящая коллекция писем

Эта коллекция писем включает в себя известные письма, написанные Блаженным Серафимом Платинским, адресованные современным корреспондентам.

Большинство настоящей коллекции состоит из 328 писем, опубликованных [онлайн](https://proza.ru/2016/10/04/1512) в октябре 2016 года. Дополнительные письма были вставлены из корпуса выдержек отца Алексея Юнга, опубликованных в *Письмах отца Серафима*.

Хотя индекс писем здесь остается согласованным с корпусом из 328, те письма, которые были вставлены из тома выдержек отца Алексея Юнга, дополнены точкой и дальнейшей нумерацией. Сноски были добавлены к письмам в местах, где дополнительный контекст может оказаться полезным. Некоторые заголовки, предоставленные в коллекции из 328, дополнены, чтобы отразить краткое, сокращенное название.
